# .github

Organization profile and shared community-health defaults for the **Panta Rhei Research Program**.

The public website is the research observatory. GitHub is the public engagement, source, formalization, contribution, and correction substrate.

## Public repositories

| Repository | Role |
|---|---|
| [**site**](https://github.com/Panta-Rhei-Research/site) | Jekyll source for panta-rhei.site, the public research observatory. |
| [**publications**](https://github.com/Panta-Rhei-Research/publications) | Publication artifacts for monographs, supplements, papers, notes, briefings, white papers, and release artifacts. |
| [**taulib**](https://github.com/Panta-Rhei-Research/taulib) | Lean 4 formalization surface. Current metrics and trusted-base details are published in the Release Manifest. |
| [**research**](https://github.com/Panta-Rhei-Research/research) | Public research workspace for scripts, notebooks, import reports, data transforms, and exploratory supporting material. |
| [**community**](https://github.com/Panta-Rhei-Research/community) | Public engagement hub for discussions, onboarding, routing, review guidance, and contribution policies. |
| [**.github**](https://github.com/Panta-Rhei-Research/.github) | Organization profile and shared community-health defaults. |

## This repository contains

- `profile/README.md`: the organization profile displayed on [github.com/Panta-Rhei-Research](https://github.com/Panta-Rhei-Research).
- Shared community-health files for code of conduct, support, security, issue templates, and pull request templates.
- Public routing defaults for Issues and Pull Requests across organization repositories.

## Engagement without endorsement

Participation does not imply endorsement of the framework.

We ask for open-research engagement: careful reading, public questions, critique, reproducibility checks, domain review, correction, infrastructure contribution, and responsible communication.
