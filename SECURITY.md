# Security Policy

If you discover a security issue that affects the public infrastructure of the Panta Rhei Research organization, please do not open a public issue.

Report it privately to:

hello@panta-rhei.site

Please include:

- the affected repository or surface;
- a clear description of the issue;
- reproduction steps if relevant;
- expected impact;
- any suggested mitigation.

We will review the report and respond as quickly as practical.
