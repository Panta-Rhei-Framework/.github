# Security Policy

If you discover a security issue that affects the public infrastructure of the Panta Rhei organization, please do **not** open a public issue.

Instead, report it privately to:

- contact@panta-rhei.site

Please include:

- the affected repository or surface
- a clear description of the issue
- reproduction steps if relevant
- expected impact
- any suggested mitigation

We will review the report and respond as quickly as practical.
